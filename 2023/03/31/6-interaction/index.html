6. Interaction/Moderation effect

2023-03-31

I still like to illustrate interaction or moderation effects with formulas as I think it is the most clear way.

Previously we examined this model with regression

Statistically, it is expressed by this regression model:
EnvironmentalKnowledge=B0+B1MediaUse+B2Education+B4gender+B5age+B6income+εEnvironmental Knowledge = B_{0} + B_{1} Media Use + B_{2} Education + B_{4} gender+ B_{5} age + B_{6} income + \varepsilonEnvironmentalKnowledge=B0​+B1​MediaUse+B2​Education+B4​gender+B5​age+B6​income+ε

The moderated regression model is EnvironmentalKnowledge=B0′+B1′MediaUse+B2′Education+B3′MediaUse×Education+B4′gender+B5′age+B6′income+ε′Environmental Knowledge = B_{0}^{\prime} + B_{1}^{\prime} Media Use + B_{2}^{\prime} Education + B_{3}^{\prime} Media Use \times Education + B_{4}^{\prime} gender+ B_{5}^{\prime} age + B_{6}^{\prime} income + \varepsilon^{\prime}EnvironmentalKnowledge=B0′​+B1′​MediaUse+B2′​Education+B3′​MediaUse×Education+B4′​gender+B5′​age+B6′​income+ε′

Sometimes it’s called interaction because the term MediaUse×EducationMedia Use \times EducationMediaUse×Education is called interaction term. But you can also see it as education moderated the relationship between media use and environmental knowledge (1). You can also see it as media use moderated the relationship between education and environmental knowledge (2).

For (1), the equation is like EnvironmentalKnowledge=B0′+(B1′+B3′Education)MediaUse+B2′Education+B4′gender+B5′age+B6′income+ε′Environmental Knowledge = B_{0}^{\prime} + (B_{1}^{\prime}+ B_{3}^{\prime} Education) Media Use + B_{2}^{\prime} Education + B_{4}^{\prime} gender+ B_{5}^{\prime} age + B_{6}^{\prime} income + \varepsilon^{\prime}EnvironmentalKnowledge=B0′​+(B1′​+B3′​Education)MediaUse+B2′​Education+B4′​gender+B5′​age+B6′​income+ε′

You can know mathmatically, this equation is the same as the prior two actually. One thing changed is the coefficient of media use is dependent on education (B1′+B3′Education)(B_{1}^{\prime}+ B_{3}^{\prime} Education)(B1′​+B3′​Education). Therefore, we call that “education moderated the relationship between media use and environmental knowledge” as education changed the coefficient.

Alternatively, (2) is also reasonable because the moderated regression model is equivalent to EnvironmentalKnowledge=B0′+B1′MediaUse+(B3′MediaUse+B2′)Education+B4′gender+B5′age+B6′income+ε′Environmental Knowledge = B_{0}^{\prime} + B_{1}^{\prime} Media Use + (B_{3}^{\prime} Media Use + B_{2}^{\prime}) Education + B_{4}^{\prime} gender+ B_{5}^{\prime} age + B_{6}^{\prime} income + \varepsilon^{\prime}EnvironmentalKnowledge=B0′​+B1′​MediaUse+(B3′​MediaUse+B2′​)Education+B4′​gender+B5′​age+B6′​income+ε′
The the coefficient of education is now B3′MediaUse+B2′B_{3}^{\prime} Media Use + B_{2}^{\prime}B3′​MediaUse+B2′​, which is dependent on media use.

To judge whether there is interaction/moderation effect or not, just refer to the p-value of the interaction term. If you are going to calculate it manually, you can mean center them first and calculated the interaction term and then treat the interaction term as one independent variable and do the OLS regression like my previous post. But for convenience, I’m going to illustrate how to do and plot this with PROCESS Macro Model 1.

I continue using the same sample data about environmental knowledge to demonstrate this.

If you have successfully installed PROCESS Macro in your SPSS…Open it through Analyze -> Regression -> PROCESS.

Recall the Model 1. W is the moderator, X is the independent variable, and Y is the dependent variable. So here I’m going to choose edu as the moderator, X is media and Y is environmental knowledge.

In PROCESS results, it is equivalent to EK=constant+B1Media+B2edu+B3Int_1+B4gender+B5age+B6income+εEK = constant + B_{1} Media + B_{2} edu + B_{3} Int\_1 + B_{4} gender+ B_{5} age + B_{6} income + \varepsilonEK=constant+B1​Media+B2​edu+B3​Int_1+B4​gender+B5​age+B6​income+ε

In options, check “Generate code for visualizing interactions.” I changed the conditioning values as well but it depends.

Then we get the results like below:

We can see the interaction term is not significant (p=0.7738). Hence, we cannot reject the null hypothesis that the coefficient of the interaction term is 0. That is, the coefficient of the interaction term is possible to be 0. We did not find any interaction/moderation effect in this model.

We can also get the code for plotting at the last.

Copy the code and open “File -> New -> Syntax”, paste and run, and you can get the graph. You can edit the graph in SPSS. It’s not very convenient, but it’s my common way.

But alternatively, you can use excel to plot the figure as long as you know what the numbers in the code means.
Actually it’s quite easy…The first column is the media use points, the second is the education, and the third is the environmental knowledge. You can see from the SPSS plot that, basically the interaction plot is plotted with 9 points. It can be plotted with excel “scatter plot（散点图）” add the numbers. Then it’s the art of Excel…

Note that there are many ways to plot interaction effect, and there are many tools online. My supervisor developed one for continuous variables’ plotting. It requires mean centering. There are also other tools. You can explore them online.
